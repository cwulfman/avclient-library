AccessCodeRequest

avclient.js

Request that an access code be sent to the user (outside of th app; e.g. email), based on Voter's input data.

new AccessCodeRequest(firstName: string, lastName: string, dob: string, idNo: string): string
Parameters
firstName (string) voter's first name field
lastName (string) voter's last name field
dob (string) voter's date of birth field
idNo (string) either the ID number the voter used before (DLN, state ID, SSN) or, if no ID number was provided, then the voter record ID number returned by the previous lookup call.
Returns
string: status (success or network error)
